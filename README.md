
# Fraud Detection & Transaction Classification using Machine Learning

This project builds a simple and explainable machine learning model using a Decision Tree classifier to detect fraudulent transactions.

The system classifies transactions into:
- Normal  
- Suspicious  
- High-Risk  

## Features
- Uses transaction amount and location as input features  
- Multi-class classification  
- Explainable AI model using Decision Tree  
- Implemented using Python and Jupyter Notebook

## Technologies
- Python  
- NumPy  
- Scikit-learn  
- Jupyter Notebook  

## Machine Learning Model
- Algorithm: Decision Tree Classifier  
- Type: Supervised Learning  
- Evaluation Metric: Accuracy  

The model is designed to be simple, efficient, and interpretable.

## Usage
Run the notebook `fraud_detection.ipynb` to train the model and view predictions.

## Output
The model predicts whether a transaction is Normal, Suspicious, or High-Risk based on input features.
